# 🗄️ Lumi Database

Este repositório contém os **scripts de banco de dados** do projeto **Lumi**, uma solução acadêmica voltada para o **monitoramento inteligente da luminosidade em estufas de cultivo de alface**.

O objetivo deste banco é armazenar, organizar e estruturar os dados necessários para o funcionamento do sistema, permitindo a futura integração com dashboards, APIs e sistemas de visualização.

---

## 🌱 Sobre o Projeto

O **Lumi** é um projeto acadêmico focado em **agricultura orgânica inteligente**, com ênfase no acompanhamento da **intensidade luminosa em estufas agrícolas**.

A proposta é coletar dados de luminosidade utilizando um **sensor LDR conectado ao Arduino**, armazenando essas informações para posterior análise e apoio à tomada de decisão do produtor.

---

## 🎯 Objetivo deste Repositório

Este repositório foi criado para armazenar os **scripts SQL responsáveis pela modelagem e estruturação do banco de dados** do projeto.

Aqui estão incluídos scripts relacionados a:

- criação de tabelas
- estrutura do banco
- relacionamento entre entidades
- armazenamento de dados do sistema
- base para integração com dashboard e backend

---

## 🧠 Finalidade do Banco de Dados

O banco de dados do Lumi tem como função principal organizar as informações coletadas e utilizadas pelo sistema, permitindo o registro de dados importantes como:

- medições de luminosidade
- informações de usuários
- possíveis registros de sensores
- histórico de monitoramento
- dados necessários para análise futura
